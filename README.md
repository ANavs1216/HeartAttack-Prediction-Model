# HeartAttack-Prediction-Model
Developed an intelligent Heart Attack Prediction Model using machine learning techniques to identify high-risk patients based on medical and lifestyle data. Implemented data preprocessing, feature engineering (including BMI and risk factors), and trained classification models to achieve reliable prediction accuracy. 

🚀 Features
📊 Data preprocessing and cleaning
🧠 Feature engineering (BMI, risk factors, etc.)
🤖 Machine Learning model training & evaluation
📈 Visualization of risk trends
⏳ Time-series analysis (future scope with LSTM)
🌐 Deployment-ready (Streamlit dashboard – upcoming)

Libraries: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, TensorFlow/Keras
Tools: Jupyter Notebook, Git, GitHub
